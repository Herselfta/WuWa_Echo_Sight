# 声骸系统规则基线

本文档用于记录声骸系统的重要统计规则，作为后续讨论与分析的统一上下文。

## 核心结论

- 副词条种类出率均等：13 种副词条出现概率相同（约 7.69%）。
- 数值档位分布不均：词条出率均等，但数值档位概率不均。双暴更容易出低档，其他属性更容易出中档。
  - 暴击率 / 暴击伤害（8 档）：明显偏向低档，最低三档合计约 75%，最高两档通常低于 3%。
  - 百分比攻击 / 防御 / 生命（8 档）：近似纺锤形，中间四档合计约 75%，最高档约 3%。
  - 固定攻击 / 固定防御（4 档）：中间两档占比最高，合计超过 80%。
- 双暴档位互相独立：暴击率和暴击伤害的档位高低互不影响。
- 双暴毕业难度高：凑齐双暴本身不易（平均出率约 8.5%，约每 12 个声骸出现一个），双高档位双暴更稀有。

## 双暴概率与决策阈值
- 开洞概率衰减：
  - 第 1 个孔是单暴时，最终达成双暴概率约 33%。
  - 前 2 个孔均无暴击/爆伤时，最终达成双暴概率约 5.45%（通常建议放弃）。
  - 前 4 个孔均无暴击/爆伤时，最终达成双暴概率为 0（不可能双暴）。
